## 5. Lighthouse 目標

本番URL（https://369web-prototype-90k-salon.vercel.app）で Lighthouse 測定。

| カテゴリ | 目標値 | 結果 |
|---|---|---|
| Performance | 80+ | [ ] |
| Accessibility | 95+ | [ ] |
| Best Practices | 90+ | [ ] |
| SEO | 90+ | [ ] |

### 測定手順
1. Chrome DevTools → Lighthouse タブ
2. Mode: Navigation / Device: Mobile
3. Categories: 全選択
4. Analyze page load

> 注: SEO は noindex 設定により警告が出るが許容（デモサイトのため）。
> 「Page is blocked from indexing」と表示されるのは設計通り。

### Lighthouse 未達時の対応
- Performance 80未満: フォント読み込み・画像最適化を確認
- Accessibility 95未満: alt属性・aria-label・コントラスト比を確認
- Best Practices 90未満: HTTPS / Console エラー / 古いライブラリを確認
